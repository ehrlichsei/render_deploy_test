About This App

This is a simple Flask application built to demonstrate deployment on Render.

The application uses Flask as the web framework and Bootstrap for styling.

Deployment Details

This app is deployed on Render, a unified cloud platform that offers free and easy deployment for web applications. Render provides built-in support for Python web applications, making it an excellent choice for Flask apps.

Technology Stack
  • Python
  • Flask
  • Bootstrap
  • Render Hosting
